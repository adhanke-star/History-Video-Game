1. **North star:** stylized-realistic 3D (modern-Total-War readability; NOT photoreal cutscene). Cohesion > any single model's realism.
2. **Architecture:** **max out three.js, single-file** — no build step; post-processing/PBR/shadows from the same jsdelivr CDN. Zero-install + saga-file portability preserved.
3. **Budget:** **free / CC0 only.** No paid packs, no commissions. Free-tier AI-gen credits OK (don't waste them).
4. **Units:** **Tripo3D** (free tier) — rigged + animated + multiview + game-ready (retopo). Primary unit pipeline. fal.ai/Rodin = static workhorse/fallback.
5. **Terrain/sky:** **Poly Haven CC0** PBR ground materials + HDRI skies.
6. **Animation:** **core set, event-mapped** — idle / march / fire / fall-die / charge / rout, played by a Three.js `AnimationMixer` driven by game state. Includes the engine animation-playback wiring.
7. **Post-processing:** **cinematic max** — bloom + SSAO + ACES tone-map + period color-grade + SMAA + depth-of-field + vignette + film grain. **MUST be perf-tiered** (see §C hardware).
8. **Battle FX:** **cinematic particles** — lingering drifting smoke banks, muzzle flash, cannon-burst, melee dust, weather particles (rain/snow), dropping rout flags, impact puffs.
9. **Portraits:** **public-domain Brady/LoC tintype photos** for real historical figures; a **tasteful tintype-framed fallback** (procedural) for player-created / generic units so there are no coverage gaps.
10. **UI/HUD:** **full art-directed pass** — engraved frames, period display type, drawn order-icons, card visual language, broadsheet newspaper menu. Bring 2D up to the 3D's level.
11. **Authenticity:** **deep period research** — public-domain references (Brady photos, regulation uniform/insignia plates, LoC) fed to Tripo multiview for museum-accurate units; logged with the existing **Verified/Inferred** tags.
12. **Run scope:** **full-blast, everything at once** (one session, all of it).
13. **Autonomy:** **full — run continuously**, splice + self-verify, report at milestones; stop only on a hard gate (build break, credits exhausted, genuine design fork).
